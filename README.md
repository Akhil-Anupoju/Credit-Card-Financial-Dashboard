# Credit Card Financial Dashboard

## Overview
The **Credit Card Financial Dashboard** is a comprehensive Power BI project designed for data analysis and visualization of credit card transactions. This project utilizes a SQL database to manage and analyze financial data, providing insights into spending patterns, trends, and key performance indicators.

## Features
- **End-to-End Project**: Complete workflow from data extraction to visualization.
- **Interactive Dashboard**: User-friendly interface with interactive charts and graphs.
- **Data Insights**: Detailed analysis of credit card usage, payments, and trends.
- **SQL Integration**: Efficient data handling and querying using SQL.

## Technology Stack
- **Power BI**: For data visualization and dashboard creation.
- **SQL Database**: To store and manage financial data.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Akhil-Anupoju/Credit-Card-Financial-Dashboard.git
   ```
2. Ensure you have Power BI Desktop installed.
3. Set up your SQL database and import the provided data schema.
4. Open the Power BI file and connect to your SQL database.

## Usage
- Launch Power BI and open the dashboard file.
- Connect to your SQL database.
- Explore the various visualizations to gain insights into credit card financial data.
